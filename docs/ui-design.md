# VoiceBuilder UI 设计方案 v2 — 简洁实用工具风格

> 定位:为普通用户设计的**伪声练习工具**。
> 目标:打开即用、一眼看懂哪里可操作、有常驻帮助说明;不追求"专业工作站"的厚重感。

---

## 1. 设计原则(与上一版的核心差异)

| 维度 | 原方案(DAW 风格) | 新方案(简洁实用) |
|---|---|---|
| 视觉基调 | 深色、强对比、品牌色浓 | 浅色/中性为主,仅 1–2 个强调色 |
| 信息密度 | 面板堆叠、细节丰富 | 留白充足、一次只聚焦一件事 |
| 操作区识别 | 需要学习、依赖图标 | 按钮/卡片有明确文字说明,常驻提示 |
| 帮助说明 | 问号抽屉、进入后才看到 | 常驻文本 + 工具卡片,随时可见 |

**三条核心原则:**
1. **主视觉只有一张图 + 一条曲线**(语谱图 + 共振峰曲线),其余都是配角。
2. **可操作区域 = 按钮或卡片**,必须有**文字标签**(不做纯图标按钮)。
3. **常驻帮助**:页面顶部/侧边放置"这是什么 / 怎么用"的 2–3 行简洁说明或卡片区。

## 3. 可操作区域清单(必须有文字标签)

| 位置 | 按钮 | 图标 | 颜色 | 功能 |
|---|---|---|---|---|
| 顶栏(左到右) | `开始录音` | ●(红色) | 红底白字,圆角 8px,主按钮 | 进入录音状态 |
| 顶栏 | `导入 WAV` | 📁 | 白底灰色边框,次按钮 | 选择本地 wav 分析 |
| 顶栏 | `清空` | ↺ | 白底灰色边框,次按钮 | 重置视图与数据 |
| 顶栏 | `帮助` | ? | 白底灰色边框,次按钮 | 打开帮助面板 |
| 共振峰卡片 | `图例项`(F0/F1/F2/F3/F4) | ●(小圆) | 曲线颜色 | 点击切换显示/隐藏 |
| 语谱图卡片 | `目标带图例` | 色块 | 浅绿/浅蓝/浅黄 | 悬停时显示区间说明 |

> 设计要点:**顶栏永远是可操作主区域**,用户不必再去角落找按钮。

## 4. 帮助说明的两种形式

### 4.1 常驻引导条(首页上方,可关闭)

> 面向首次用户,简洁一两行,不会喧宾夺主。

```
💡 对着麦克风发声,观察下方两条曲线。想练习"伪女声"时,F0 落在 180-250Hz 区间效果更好。
   [了解更多 ↗]   [隐藏此条 ×]
```

- 使用浅色背景 + 柔和边框,与正文有区分但不刺眼。
- 关闭后不再弹出(可存 localStorage)。

### 4.2 帮助面板(点击"?"按钮后弹出,侧抽屉/浮层)

内容结构(每节不超过 100 字):

1. **如何开始** — "点击顶部"开始录音",或"导入 WAV"选择已有音频。"
2. **声音频谱(Spectrogram)** — "横向是时间,纵向是频率。颜色越亮代表该频率能量越大。"
3. **共振峰(Formants)** — "F0 是基频(决定声音高低),F1/F2 决定音色。绿色区域是目标区间。"
4. **目标区间含义** — "伪女声参考:F0 180–250 Hz, F1 400–700 Hz, F2 1500–2300 Hz。"
5. **常见问题** — "麦克风没反应?请检查浏览器权限;曲线抖动?说话持续一点更稳定。"

## 5. 视觉规范

### 5.1 颜色(浅色主题为主,深色为可切换项)

| 用途 | 颜色(浅色) | 说明 |
|---|---|---|
| 页面背景 | `#FAFBFC` | 中性浅灰,不刺眼 |
| 卡片背景 | `#FFFFFF` | 纯白 + 浅阴影 |
| 卡片边框 | `#E5E7EB` | 细 1px,无强对比 |
| 主强调色(录音/主按钮) | `#E23E57`(红) | 仅用于"开始录音"等关键主按钮 |
| 命中态(目标区) | `#3DDC97`(薄荷绿) | 曲线段、区间背景、色块图例 |
| 次按钮文本/边框 | `#4B5563` / `#D1D5DB` | 中性灰,不抢主视觉 |
| 正文文字 | `#1F2937` | 保证 AA 对比度 |
| 次要文字/说明 | `#6B7280` | 不抢重点 |
| 语谱图底色(浅色模式) | `#F3F4F6` | 热图在浅色底也可读 |
| 语谱图渐变色板 | `Viridis` 或 `Plasma`(浅→深,色彩柔和,非彩虹) | 色盲友好 |

> 深色模式作为**可选设置**(默认给浅色),提供右上角切换图标 `🌓`。

### 5.2 字号与间距

| 元素 | 字号 | 字重 |
|---|---|---|
| 产品名(顶栏) | 18 px | 600 |
| 卡片标题 | 16 px | 600 |
| 卡片说明文字 | 13 px | 400 |
| 正文/按钮文字 | 14 px | 500 |
| 数值(底部状态栏) | 13 px 等宽 | 500 |
| 刻度/坐标 | 12 px | 400 |

**间距(8pt grid):**
- 顶栏上下 padding:`16 px`
- 卡片内边距:`16 px`
- 卡片之间间距:`24 px`
- 主内容左右留白:`≥ 32 px`(大屏更多)
- 按钮高度:`40 px`(移动端 48 px)
- 按钮水平 padding:`16 px`
- 圆角:`8 px`(按钮/卡片一致)

### 5.3 阴影层级

| 层级 | 取值 | 用途 |
|---|---|---|
| L1(默认卡片) | `0 1px 2px rgba(0,0,0,0.04), 0 1px 3px rgba(0,0,0,0.06)` | 普通卡片 |
| L2(悬停/抽屉) | `0 4px 12px rgba(0,0,0,0.08)` | 悬停态/帮助抽屉 |
| L3(弹出/模态) | `0 10px 25px rgba(0,0,0,0.12)` | 模态框 |

- **ECharts 主题**:使用默认 `light`(不使用 `dark`)

- **目标区间**:`markArea` 在 F0/F1/F2 目标区间填充 `rgba(61, 220, 151, 0.15)`
- **legend**:卡片右上角,点击可切换显示/隐藏


- 不干扰主视图,仅作为兜底数值展示
- 等宽字体让 Hz 数值稳定

## 7. 录音状态与反馈(让用户知道"发生了什么")

| 状态 | 视觉表现 |
|---|---|
| **空闲** | "● 开始录音"按钮显示为红色,其余为灰色次按钮 |
| **请求麦克风权限** | 按钮变为 `…授权中`,浅灰,禁用点击(防止重复触发) |
| **录音中** | 按钮变为 `■ 停止录音`,红色加深,按钮有柔和呼吸动画;右侧 `00:42` 计时;页面顶部出现小红点"●"状态指示 |
| **分析中(导入 WAV)** | 共振峰卡片区域显示 `正在分析…` 骨架屏/进度条 |
| **错误(麦克风不可用)** | 按钮变为 `⚠ 麦克风不可用,点击重试`,琥珀色背景,边框加粗 |
| **命中目标区间** | 对应共振峰曲线段 + 语谱图区间变为绿色高亮;底部状态栏对应数值变为绿色 |

## 11. 与上一版的差异摘要

| 方面 | 上一版 | 本版 |
|---|---|---|
| 主题 | 深色 DAW 风格 | 浅色工具风格(深色为可选) |
| 操作入口 | 角落小圆按钮 | 顶栏主按钮 + 文字标签,一眼看到 |
| 帮助说明 | 隐藏在问号抽屉 | 常驻引导条(2-3 行)+ 帮助面板 |
| 强调色 | 多种高饱和品牌色 | 红(录音)+ 绿(命中),其余中性灰 |
| 信息密度 | 卡片堆叠,细节多 | 留白充足,一次一件事 |
| 语谱图色板 | 深蓝→黄经典 | Viridis/Plasma,色盲友好 |
| 组件感觉 | "有门槛的专业工具" | "随手可用的日常工具" |
