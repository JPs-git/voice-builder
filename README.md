
# Voice Builder

实时分析语音音高, 共振峰, 专为伪音练习设计.

<img alt="Logo" src="assets/logo-with-text.png" width="80%">

## Quick Start

克隆本仓库
```bash
  git clone https://github.com/JPs-git/voice-builder.git
  cd voice-builder
```
安装依赖启动开发服务器
```bash
  npm install
  npm run dev
```

## Features

- 实时显示音高、共振峰
- 回放自己的声音
- 支持上传wav文件分析
