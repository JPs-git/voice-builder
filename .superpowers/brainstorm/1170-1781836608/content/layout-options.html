声谱图 vs 共振峰图谱 — 布局方案

选一个你最倾向的布局方式，或选多个表示"都想要"

A

叠加显示 (Praat 风格)

共振峰轨迹 (F0–F4) 直接画在声谱图上，时间轴共用，一目了然。

优点

  • 紧凑，信息密度高
  • 符合语音学界习惯
  • 时间对齐无歧义

缺点

  • 初学者可能觉得杂乱
  • 声谱图能量可能掩盖共振峰线
B

上下分离视图

上方声谱图，下方单独画共振峰折线图 (纵轴 Hz)，时间轴同步。

优点

  • 干净清晰，互不干扰
  • 共振峰数值容易读取
  • 适合初学者理解

缺点

  • 占更多屏幕空间
  • 视线需要上下移动
C

可切换标签

通过 Tab/开关在"声谱图"和"共振峰"视图之间切换，同一块区域。

优点

  • 最大利用屏幕空间
  • 专注当前视图

缺点

  • 无法同时看到两者
  • 快速对比不方便
D

叠加 + 可切换分离

默认叠加显示，但提供一个按钮一键切换到上下分离视图。

优点

  • 兼顾两种需求
  • 用户可自由选择

缺点

  • 实现复杂度增加
  • 开发时间增加

在浏览器里点击选择，或者在终端告诉我你的想法都可以。
